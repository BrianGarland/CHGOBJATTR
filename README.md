**What is CHGOBJATTR?** 

CHGOBJATTR is a utility to set various object attributes using the QLICOBJD API. 


**Download with RELIC:**
 
```
RELICGET PLOC('https://github.com/BrianGarland/CHGOBJATTR/archive/master.zip') PDIR('CHGOBJATTR') PNAME(#CHGOBJATTR)
```
